# 1레벨의 코드 수정

# 프로세스

- [ ] 구매 금액 입력
- [ ] 구입 버튼 클릭
- [ ] 구매 결과 출력
- [ ] 당첨번호와 보너스번호 입력
- [ ] 결과 확인하기 버튼 클릭
- [ ] 결과 모달 출력 + 결과 출력
- [ ] x버튼을 누르면 입력했던 결과 유지
- [ ] 다시 하기 누르면 입력 초기화

# 구매 금액 입력

- [ ] input을 두고 값을 입력받는다. (dom 찾기는 class이름으로)
- [ ] 해당 태그 안에 있는 입력값을 컨트롤러 클래스의 인자로 넘긴다.
- [ ] 컨트롤러는 render(view)를 호출한다.
